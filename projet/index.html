Qu'est-ce que le ballon d'or ?

Le ballon d'or est la récompense individuelle ultime pour tous joueurs professionnel de football.
Ce trophée est décerné tous les ans lors d'une cérémonie organisée par France Football. Cette récompense est attribuée suivant plusieurs critères,
tels que le nombre de buts marqués ainsi que le nombre de passes décivives.
